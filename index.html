열정 넘치는 신입 웹 퍼블리셔로, 혁신적인 디자인과 사용자 경험을 통해 웹을 빛나게 만드는 것을 사랑합니다.

ABOUT

안녕하세요 저는 신입 웹 퍼블리셔 박성하입니다.
저는 웹 퍼블리셔로서 HTML, CSS, JavaScript를 능숙하게 다루며,
최적화된 사용자 경험을 제공하기 위해 웹 표준과 접근성을 고려하여 웹 사이트를 디자인하고 구현합니다.
새로운 기술과 트렌드를 끊임없이 학습하며, 사용자들에게 최고의 웹 경험을 제공하기 위해 노력합니다.
함께하는 프로젝트에서 뛰어난 웹 퍼블리싱 능력을 발휘하여 결과물을 높은 수준으로 이끌어 나갈 것입니다.
